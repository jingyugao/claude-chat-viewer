# 问候Skill

当这个skill被调用时，请执行以下步骤：

1. **询问用户名称**：使用AskUserQuestion工具询问用户的名字
   - 提供几个常见的中文名字作为选项
   - 允许用户选择"Other"来输入自定义名称

2. **友好问候**：获取用户名称后，用温馨友好的方式向用户问好
   - 使用中文进行问候
   - 可以包含一些积极正面的问候语
   - 让用户感到欢迎和舒适

示例问候语：
- "你好，[用户名]！很高兴见到你！😊"
- "嗨，[用户名]！祝你今天过得愉快！"
- "欢迎，[用户名]！有什么我可以帮助你的吗？"
